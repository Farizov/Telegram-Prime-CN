##这是Telegram Prime功能的简短列表：

- 帐户管理：
   *具有帐户品牌的多线程帐户管理：设置照片，状态，名称，生物，2FA和20个功能。
 - 帐户自动注册：
   *使用12个以上的SMS服务注册帐户，并在多个流中有特殊折扣（多线程）。您只需要调整设置并选择要创建多少个帐户，所有其他操作软件都将为您完成。
 - 检查手机号码是否有电报：
   *在检查器的帮助下，您可以检查手机号码是否在电报中的存在，从而进一步增加基地。
 -  ID的记录：
   *从任何电报组中收集听众。过滤数据库，并完全获得目标受众。
 - 大量发送消息（PM和组）：
   *您可以通过ID，用户名或电话号码向以前收集的受众收集到的广告消息的大量邮件，也可以将消息发送到聊天。您还可以批量发送语音消息，照片，视频和其他文件。通过@PostBot或同等数字发送消息。
- 电话应答机：
   *随着自动回答器的运行，您的机器人将全天候回答传入的消息。
 - 解析/收集活跃的受众：
   *使用收集聊天用户的功能，从任何电报组中收集现场受众。小组解析器还可以与私人链接和频道一起使用开放评论。
 - 在给定的地理位置内解析/收集受众。
   *允许通过扫描任何对象的地理位置（即超市）来收集电报用户
 - 邀请转介到机器人：
   *通过转介链接邀请您的帐户。
 - 小组记录：
   *将收集到的目标受众添加到您的电报组（聊天）中，通过活动过滤。
 - 作弊民意调查，发布反应和投票。
   *参加匿名民意调查，并在频道中的帖子上提出不同的反应。在bot中的频道/组/裁判中的作弊帐户。
 - 帖子克隆器：
   *允许您克隆任何人的频道帖子。
 - 组和频道克隆器：
   *使用其他模块，您可以精确或更改任何关键字为唯一的通道。
 - 软件已自动更新：
   *我们的团队正在不断监视市场和电报源代码更新，并为我们的软件实施新功能。
 - 多语言支持团队：
   *我们有多语言的在线支持团队，他们准备回答您的所有问题并为24/7提供帮助。
